Dotfiles ⚈ ̫⚈
=============

Install
-------
```bash
$ git clone git@github.com:cblakkan/dotfiles.git
$ ./dotfiles/bin/dotfiles
```

Update configs
--------------
```bash
$ dotfiles
```
